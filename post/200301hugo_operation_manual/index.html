Hugo的基本操作

2020年3月1日
| hugo
| 阅读
文章目录

本篇博客用来记录搭建hugo后，发布新博客的一些基础操作，一切操作基于win10+hugo+git，为了方便快捷，也可以根据本博客自己写一个脚本，一键发布。

1 win+r接着cmd进入命令行界面，接着cd进入myblog目录

2 创建新博客

hugo new post\hugo_operation_manual.md

来生成新的博客，博客名为hugo_operation_manual

3 接下来在本地渲染，预览效果

hugo server -t m10c --buildDrafts

其中m10c为主题目录名，在浏览器中打开

localhost:1313

访问。

4 渲染，准备上传GitHub

渲染

hugo --theme=m10c --baseUrl="https://xzyu0106.github.io" --buildDrafts

接着cd进入public目录，

cd public

输入

git add .

再输入以下命令作为提交声明

git commit -m "提交"

下面这一步可以忽略

git remote add origin https://github.com/xzyu0106/xyzu0106.github.io.git

5 上传至GitHub

git push -u origin master
